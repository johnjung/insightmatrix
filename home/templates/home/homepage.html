Insight Matrix

Insight Matrix is a tool for finding patterns in qualitative data, bringing analytic rigor to qualitative research. It allows researchers and project teams to set up a matrix of ideas and score them against each other. It sorts like ideas together, allowing users to create clusters of the sorted ideas to use as frameworks.
